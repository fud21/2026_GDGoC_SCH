# GDGoC SCH 프론트엔드

투자 교육·모의투자 플랫폼의 모바일 우선 React SPA입니다. 전체 설치 및 DB/백엔드
실행 순서는 루트 [README](../README.md)를 참고하세요.

## 명령어

```bash
npm install
npm run dev
npm run lint
npm run build
```

개발 서버는 `/api` 요청을 `http://localhost:4000`의 Express 백엔드로 프록시합니다.

## 구현 범위

- 인증 및 투자 성향 온보딩
- 홈의 레벨/XP 현황
- 챕터·레슨·퀴즈·용어사전
- 모든 화면의 교육 목적 고지

모의투자와 AI 어시스턴트는 후속 단계입니다.

### 로컬 실행 전 확인 사항

회원가입과 로그인 기능을 사용하려면 프론트엔드와 백엔드뿐만 아니라
MySQL 데이터베이스도 실행되어 있어야 합니다.

1. `backend/.env.example`을 복사해 `backend/.env` 파일을 생성합니다.
2. `.env`에 로컬 MySQL 접속 정보를 입력합니다.
3. 프로젝트의 데이터베이스 스키마를 MySQL에 적용합니다.
4. 프론트엔드와 백엔드를 각각 실행합니다.

백엔드 서버가 실행되더라도 데이터베이스가 연결되지 않으면
회원가입 과정에서 오류가 발생할 수 있습니다.
