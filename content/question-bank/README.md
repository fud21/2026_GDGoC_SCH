# Question bank review policy

이 디렉터리의 문항은 팀에서 전달받은 프로토타입 원본을 보존한 **검수 대기 콘텐츠**입니다. 현재 사용자 화면이나 채점 API에는 자동으로 연결되지 않습니다.

## Inventory

- `stock_quiz.json`: 일반 학습 문항 302개
- `exam_questions.json`: 6개 등급 시험 문항 103개

## 배포 전 필수 검수

1. 금융·세금·규제·제도 관련 내용은 배포 시점의 공식 출처로 다시 확인합니다.
2. 종목 매수·매도 권유, 수익 보장, 과도하게 단정적인 문구를 제거합니다.
3. 저작권, 상표, 인용 출처 및 데이터 라이선스를 확인하고 기록합니다.
4. 정답과 해설을 금융 지식 검수자 1명 이상이 교차 검토합니다.
5. `npm run validate:content`로 ID 중복, 선택지·정답, 필수 필드를 검사합니다.

자동 검증은 구조적 오류만 찾습니다. 사실 정확성이나 교육 적합성을 보증하지 않습니다.
